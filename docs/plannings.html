Planning Objects

When you enroll for a course or study, it can be useful to know when it takes place. There are two ways you can do this, either via a nested query on the course or study object itself, or by calling planningsForCourses or planningsForStudies.

The two dedicated queries take one parameter, either courses or studies, which indicates what courses/studies you want the planning objects for. These parameters take the IDs of the objects of interest, for example, you can request the planning for studies 3, 5 and 7 like so.

{
    planningsForStudies(studies: [3, 5, 7]){
        start_time
        end_time
    }
}

Each planning object has a start and end time, and some other information that might be useful. As always, you can select the required field yourself. A full description of fields is available in the API docs
